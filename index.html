To grab the value of a key that is nested in an object, you must scope into that object. We scope into objects with {.section objectName} and close the reference with {.end}. Once scoped, any key/value pair inside that object is referenced with {keyName}. Here's an example: 

					{.section someObject}
					  <p>{someKey}</p>
					{.end}
